# Задание 2: Многомодульный проект на Gradle

## Описание задания

Необходимо создать многомодульный проект на Gradle, который демонстрирует работу с модульной архитектурой.

## Требования

1. Создать корневой проект с файлом `settings.gradle`
2. Создать минимум 3 модуля:
   - `utils` - модуль с утилитными классами
   - `service` - модуль с бизнес-логикой, зависящий от `utils`
   - `app` - главный модуль приложения, зависящий от `service` и `utils`
3. Настроить зависимости между модулями через `build.gradle`
4. Создать классы в каждом модуле с комментариями на русском языке
5. Написать тесты для проверки функциональности

## Решение

Проект создан с использованием Gradle и включает три модуля с четкой иерархией зависимостей.
