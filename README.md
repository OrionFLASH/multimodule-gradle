# Многомодульный проект Gradle

## Описание проекта

Данный проект представляет собой учебный многомодульный проект, созданный с использованием Gradle. Проект демонстрирует организацию кода в модульной архитектуре с четким разделением ответственности между модулями.

## Техническое задание

Создать многомодульный проект на Gradle со следующей структурой:
- Корневой проект с файлом `settings.gradle`
- Модуль `utils` - утилитные классы
- Модуль `service` - бизнес-логика, зависящая от `utils`
- Модуль `app` - главное приложение, зависящее от `service` и `utils`

## Структура проекта

```
multimodule-gradle/
├── settings.gradle         # Настройки проекта и список модулей
├── build.gradle            # Родительский build.gradle
├── utils/                  # Модуль утилит
│   ├── build.gradle
│   └── src/
│       ├── main/java/ru/netology/utils/
│       │   └── NumberUtils.java
│       └── test/java/ru/netology/utils/
│           └── NumberUtilsTest.java
├── service/                # Модуль сервисов
│   ├── build.gradle
│   └── src/
│       ├── main/java/ru/netology/service/
│       │   └── MathService.java
│       └── test/java/ru/netology/service/
│           └── MathServiceTest.java
└── app/                    # Главный модуль
    ├── build.gradle
    └── src/
        └── main/java/ru/netology/app/
            └── Main.java
```

## Описание решения

### settings.gradle

Определяет корневой проект и список всех модулей:
- `utils`
- `service`
- `app`

### Родительский build.gradle

Содержит общие настройки для всех модулей:
- Применение Java плагина
- Версия Java: 11
- Кодировка: UTF-8
- Зависимости: JUnit 5 для тестирования

### Модуль utils

Содержит утилитные классы для работы с числами.

**Класс NumberUtils:**
- `isEven(int number)` - проверяет, является ли число четным
- `factorial(int n)` - вычисляет факториал числа
- `gcd(int a, int b)` - вычисляет наибольший общий делитель

### Модуль service

Содержит бизнес-логику приложения. Зависит от модуля `utils`.

**Класс MathService:**
- `average(int a, int b)` - вычисляет среднее арифметическое
- `power(int base, int exponent)` - возводит число в степень
- `checkEven(int number)` - проверяет четность числа
- `calculateFactorial(int n)` - вычисляет факториал

### Модуль app

Главный модуль приложения, демонстрирующий использование всех модулей.

**Класс Main:**
- Точка входа в приложение
- Демонстрирует работу с утилитами и сервисами

## Используемые инструменты

- **Gradle** - система сборки и управления зависимостями
- **Java 11** - язык программирования
- **JUnit 5** - фреймворк для тестирования

## Классы и функции

### ru.netology.utils.NumberUtils

Утилитный класс для работы с числами.

**Методы:**
- `public static boolean isEven(int number)` - проверка четности числа
- `public static long factorial(int n)` - вычисление факториала
- `public static int gcd(int a, int b)` - вычисление НОД

### ru.netology.service.MathService

Сервис для выполнения математических операций.

**Методы:**
- `public double average(int a, int b)` - среднее арифметическое
- `public long power(int base, int exponent)` - возведение в степень
- `public boolean checkEven(int number)` - проверка четности
- `public long calculateFactorial(int n)` - вычисление факториала

### ru.netology.app.Main

Главный класс приложения.

**Методы:**
- `public static void main(String[] args)` - точка входа

## Сборка проекта

Для сборки проекта выполните команду:

```bash
./gradlew build
```

Для запуска приложения:

```bash
./gradlew :app:run
```

Для запуска тестов:

```bash
./gradlew test
```

Для создания исполняемого JAR:

```bash
./gradlew :app:jar
```
